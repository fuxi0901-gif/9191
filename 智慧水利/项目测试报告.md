### 1.3 详细搭建步骤

#### 1.3.1 克隆代码
```bash
git clone <项目地址>
cd 智慧水利
```

#### 1.3.2 后端环境搭建

**步骤 1：创建虚拟环境**
```powershell
# Windows PowerShell
py -3 -m venv venv
.\venv\Scripts\Activate.ps1
```

**如果提示"禁止运行脚本"：**
```powershell
Set-ExecutionPolicy -Scope CurrentUser RemoteSigned
# 输入 Y 确认，然后重新激活
.\venv\Scripts\Activate.ps1
```

**步骤 2：安装依赖**
```powershell
pip install -r requirements.txt
```

依赖列表：
```
fastapi>=0.104.0          # Web 框架
uvicorn[standard]>=0.24.0 # ASGI 服务器
python-multipart>=0.0.6   # 文件上传
pydantic>=2.0             # 数据验证
python-dotenv>=1.0.0      # 环境变量
sqlalchemy>=2.0           # ORM
pymysql>=1.0.0            # MySQL 驱动
redis>=4.5.0              # Redis（预留）
sentence-transformers>=2.2.0  # 嵌入模型
chromadb>=0.4.0           # 向量数据库
PyPDF2>=3.0.0             # PDF 文本提取
python-docx>=1.1.0        # DOCX 文本提取
numpy>=1.21.0             # 数值计算
requests>=2.28.0          # HTTP 请求
python-jose[cryptography]>=3.3.0 # JWT
bcrypt>=4.0               # 密码哈希
```

**步骤 3：配置环境变量**

项目根目录 `.env`（大模型配置）：
```env
HF_ENDPOINT=https://hf-mirror.com
LLM_PROVIDER=deepseek
DEEPSEEK_API_KEY=your_deepseek_api_key_here
DEEPSEEK_API_BASE=https://api.deepseek.com
DEEPSEEK_MODEL=deepseek-chat
KIMI_API_KEY=your_kimi_api_key_here
OLLAMA_API_BASE=http://localhost:11434
```

`backend/.env`（数据库配置）：
```env
MYSQL_USER=root
MYSQL_PASSWORD=your_password    # ⚠️ 密码含 @ 需处理，见缺陷分析
MYSQL_HOST=127.0.0.1
MYSQL_PORT=3306
MYSQL_DB=hydrau_db
```

**步骤 4：初始化数据库**

如果使用 MySQL：
```sql
CREATE DATABASE hydrau_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

启动时会自动创建表结构，无需手动执行 SQL。MySQL 不可用时自动回退 SQLite。

**步骤 5：启动后端**
```powershell
python -m uvicorn backend.main:app --host 127.0.0.1 --port 8080 --reload
```

验证：访问 http://127.0.0.1:8080/ 应返回：
```json
{"status":"ok","message":"水利工程智能问答系统 API 运行中"}
```

API 文档：http://127.0.0.1:8080/docs

#### 1.3.3 前端环境搭建

**步骤 1：进入前端目录安装依赖**
```powershell
cd frontend
npm install
```

**步骤 2：启动开发服务器**
```powershell
npm run dev
```

默认地址：http://localhost:3000

Vite 自动代理 `/api` → `http://127.0.0.1:8080`。

**步骤 3：生产构建**
```powershell
npm run build
```

产物在 `frontend/dist/`。

#### 1.3.4 Docker 一键部署（可选）

`docker-compose.yml` 已配置：
```bash
docker-compose up -d
```

服务：
- 前端：`http://localhost:80`
- 后端：`http://localhost:8080`
- MySQL：`localhost:3306`
- ChromaDB：`localhost:8000`

### 5.3 修复 B001（密码@问题）的代码建议

在 [backend/config.py](file:///d:/桌面/water_Agent/智慧水利/backend/config.py#L25-L28) 修改为：

```python
from urllib.parse import quote

SQLALCHEMY_DATABASE_URI: str = (
    f"mysql+pymysql://{MYSQL_USER}:{quote(MYSQL_PASSWORD)}"
    f"@{MYSQL_HOST}:{MYSQL_PORT}/{MYSQL_DB}?charset=utf8mb4"
)
```

这样密码里的 `@` 会自动转成 `%40`，不会导致 URL 解析错误。
